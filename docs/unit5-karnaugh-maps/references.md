1. [Karnaugh map](https://en.wikipedia.org/wiki/Karnaugh_map) - Wikipedia - Comprehensive guide to K-map theory including grouping rules, Gray code ordering, and simplification techniques for 2-6 variable functions. Essential reference for visual Boolean minimization.

2. [Prime implicant](https://en.wikipedia.org/wiki/Implicant) - Wikipedia - Detailed explanation of implicants, prime implicants, and essential prime implicants with formal definitions. Critical for understanding optimal grouping in K-maps.

3. [Gray code](https://en.wikipedia.org/wiki/Gray_code) - Wikipedia - Overview of Gray code properties and why it's used in K-map labeling to ensure adjacent cells differ by one variable. Explains the mathematical basis for K-map adjacency.

4. Digital Design (6th Edition) - M. Morris Mano, Michael D. Ciletti - Pearson - Chapter 4 provides extensive K-map coverage with systematic approaches to finding minimal SOP and POS expressions including don't care handling.

5. Fundamentals of Logic Design (7th Edition) - Charles H. Roth Jr., Larry L. Kinney - Cengage - Chapter 5 covers K-maps thoroughly with numerous worked examples for 2, 3, 4, and 5 variable functions.

6. [Karnaugh Map Tutorial](https://www.electronics-tutorials.ws/boolean/bool_9.html) - Electronics Tutorials - Interactive K-map tutorial with step-by-step grouping examples for SOP simplification including wraparound and corner grouping.

7. [K-Map Simplification](https://www.allaboutcircuits.com/textbook/digital/chpt-8/karnaugh-maps/) - All About Circuits - Comprehensive guide to K-map techniques covering group sizes, overlapping groups, and finding all prime implicants.

8. [4-Variable K-Map Examples](https://www.geeksforgeeks.org/k-mapkarnaugh-map/) - GeeksforGeeks - Collection of 4-variable K-map problems with solutions demonstrating various grouping patterns and edge cases.

9. [K-Map with Don't Cares](https://www.allaboutcircuits.com/textbook/digital/chpt-8/larger-karnaugh-maps/) - All About Circuits - Guide to handling don't care conditions in K-maps with examples showing how to exploit them for simpler expressions.

10. [K-Map Solver](https://www.charlie-coleman.com/experiments/kmap/) - Charlie Coleman - Interactive K-map tool for 2-4 variables that shows groupings and generates minimized expressions, useful for checking manual work.
