<h1 style="color: #5A3EED !important; border-bottom: 3px solid #5A3EED; padding-bottom: 0.4rem; font-weight: 800; margin-bottom: 1.5rem;">Glossary: Number Systems</h1>

<p style="color: #555; line-height: 1.85; font-size: 1.05rem; margin-bottom: 2rem;">
Key terms and definitions for Unit 1. Definitions follow ISO 11179 metadata registry standards.
</p>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">A</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Analog Signal</strong> — A continuous signal that can take any value within a range, such as a smooth audio waveform from a microphone, in contrast to the discrete values used in digital systems.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">B</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Base (Radix)</strong> — The number of unique symbols used in a positional number system, which determines the multiplier for each position (e.g., 2 for binary, 8 for octal, 10 for decimal, 16 for hexadecimal).</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Binary Addition</strong> — The process of adding two binary numbers using the rules <span class="arithmatex">\(0+0=0\)</span>, <span class="arithmatex">\(0+1=1\)</span>, <span class="arithmatex">\(1+0=1\)</span>, and <span class="arithmatex">\(1+1=10\)</span> (0 with a carry of 1), following the same columnar approach as decimal addition.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Binary Division</strong> — The process of dividing binary numbers using the long division algorithm, simplified by the fact that only 0 and 1 are possible quotient digits at each step.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Binary Multiplication</strong> — The process of multiplying binary numbers where each partial product is either zero or the multiplicand itself (shifted appropriately), since the only possible multiplier digits are 0 and 1.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Binary Number System</strong> — A base-2 positional number system using only two symbols (0 and 1) that maps directly to the two voltage states in digital circuits, making it the native language of computers.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Binary Subtraction</strong> — The process of subtracting one binary number from another using the rules <span class="arithmatex">\(0-0=0\)</span>, <span class="arithmatex">\(1-0=1\)</span>, <span class="arithmatex">\(1-1=0\)</span>, and <span class="arithmatex">\(0-1=1\)</span> (with a borrow from the next higher position).</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Bit (Binary Digit)</strong> — The fundamental unit of digital information, representing a single 0 or 1; the term is a portmanteau of "binary" and "digit."</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Borrow</strong> — A value taken from the next higher bit position during binary subtraction when a 0 must be subtracted by a 1, analogous to borrowing in decimal subtraction.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Byte</strong> — A unit of digital data comprising 8 bits (2 nibbles) that can represent 256 distinct values (0–255 unsigned) and serves as the standard addressable unit of memory in most computer architectures.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">C</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Carry</strong> — A value generated during binary addition when the sum of bits in a column equals or exceeds 2, which is propagated to the next higher bit position.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Conversion (Base Conversion)</strong> — The process of translating a number from one base (radix) to another, using techniques such as repeated division, weighted positional expansion, or direct binary grouping shortcuts.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">D</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Decimal Number System</strong> — The base-10 positional number system using ten symbols (0 through 9), where each position represents a power of 10; the familiar number system used in everyday human arithmetic.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Digital Signal</strong> — A discrete signal that represents information using only two distinct states (typically high voltage for logic 1 and low voltage for logic 0), offering superior noise immunity over analog signals.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Digital System</strong> — An electronic circuit that processes information using discrete signal levels rather than continuous values, forming the basis of all modern computing devices.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">E</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">End-Around Carry</strong> — A correction technique used in one's complement addition where the carry-out from the most significant bit is added back to the least significant bit to obtain the correct result.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">H</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Hexadecimal Number System</strong> — A base-16 positional number system using sixteen symbols (0–9 and A–F, where A=10 through F=15), where each hex digit corresponds to exactly four binary digits, making it the preferred shorthand for binary data.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">L</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Least Significant Bit (LSB)</strong> — The rightmost bit in a binary number, which contributes the smallest positional value (either 0 or 1) to the overall number.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">M</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Most Significant Bit (MSB)</strong> — The leftmost bit in a binary number, which contributes the largest positional value; in signed number representations, the MSB typically indicates the sign of the number.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">N</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Nibble</strong> — A unit of digital data consisting of 4 bits that can represent 16 distinct values (0–15 in decimal, or 0–F in hexadecimal); each hexadecimal digit corresponds to exactly one nibble.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Noise Immunity</strong> — The ability of a digital signal to maintain its correct logical interpretation despite small variations in voltage, a key advantage of digital over analog signal representation.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">O</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Octal Number System</strong> — A base-8 positional number system using eight symbols (0 through 7), where each octal digit corresponds to exactly three binary digits; used in Unix file permissions and some assembly language contexts.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">One's Complement</strong> — A signed number representation in which negative numbers are formed by inverting (flipping) all bits of the corresponding positive number; it has the drawback of two representations for zero.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Overflow</strong> — An arithmetic error condition that occurs when the result of an operation is too large (positive) to be represented within the available bit width, detected in two's complement when the carry into the MSB differs from the carry out of the MSB.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">P</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Positional Notation</strong> — A numeral system in which the value of each digit is determined by both the digit itself and its position within the number, with each position representing a successive power of the base.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">R</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Radix Point</strong> — The symbol (such as a decimal point in base 10 or a binary point in base 2) that separates the integer portion of a number from its fractional portion in positional notation.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Repeated Division Method</strong> — A conversion technique for translating a decimal integer to another base by repeatedly dividing by the target base and recording remainders, which are then read in reverse order.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">S</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Sign Extension</strong> — The technique of preserving the numerical value of a signed binary number when increasing its bit width by replicating the sign bit (0s for positive, 1s for negative) into the additional higher-order positions.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Sign-Magnitude Representation</strong> — A signed number encoding where the most significant bit indicates the sign (0 for positive, 1 for negative) and the remaining bits represent the magnitude, mirroring how humans write signed numbers but having two representations of zero.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Signed Number</strong> — A binary number encoding that can represent both positive and negative values using schemes such as sign-magnitude, one's complement, or two's complement.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">T</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Two's Complement</strong> — The dominant signed number representation in modern digital systems, formed by inverting all bits of a number and adding 1, which provides a unique zero and enables addition and subtraction to use the same hardware.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Two's Complement Addition</strong> — The process of adding two signed numbers in two's complement representation, which works identically to unsigned binary addition with the carry out of the MSB discarded.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Two's Complement Subtraction</strong> — The process of subtracting one signed number from another by adding the two's complement (bitwise inversion plus 1) of the subtrahend, converting subtraction into addition.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">U</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Underflow</strong> — An arithmetic error condition that occurs when the result of an operation is too small (too negative) to be represented within the available bit width for the given signed number format.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Unsigned Number</strong> — A binary number representation that encodes only non-negative values, where all bits contribute to the magnitude, giving an <span class="arithmatex">\(n\)</span>-bit number the range 0 to <span class="arithmatex">\(2^n - 1\)</span>.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">W</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Weighted Positional Expansion</strong> — A conversion technique for translating a number from another base to decimal by multiplying each digit by its positional power of the base and summing the results.</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Word</strong> — A processor-specific unit of data representing the natural data size for a particular architecture (e.g., 32 bits or 64 bits), affecting memory addressing, register width, and arithmetic precision.</p>

</div>

<div style="background: #EEF4FF; border: 2px solid #A8C8FF; border-radius: 12px; padding: 28px 28px 12px 28px; margin: 1.5rem 0; box-shadow: 0 2px 8px rgba(90,61,237,0.07);">

<p style="color: #1565C0; font-weight: 700; font-size: 1.08rem; margin-top: 0; margin-bottom: 18px;">Z</p>

<p style="margin-bottom: 1rem; line-height: 1.75; color: #333;"><strong style="color: #5A3EED;">Zero Extension</strong> — The technique of increasing the bit width of an unsigned binary number by padding higher-order positions with zeros, preserving the original value.</p>

</div>
