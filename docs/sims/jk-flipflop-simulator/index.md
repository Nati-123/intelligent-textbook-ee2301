# JK Flip-Flop Simulator

An interactive JK flip-flop demonstrating set, reset, hold, and toggle operations.

## Overview

This MicroSim shows the JK flip-flop which extends SR functionality by defining the previously invalid state as a toggle operation.

## Learning Objectives

- **Understand** JK flip-flop operations including toggle (Bloom Level: L2)
- **Apply** JK flip-flop for counter design (Bloom Level: L3)

## How to Use

1. **Set** the J and K inputs
2. **Click "Clock Pulse"** to trigger the flip-flop
3. **Observe** the four operations: Hold, Reset, Set, Toggle
