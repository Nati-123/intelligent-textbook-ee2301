<h2 style="color: #5A3EED !important; border-bottom: 2px solid #5A3EED; padding-bottom: 0.3rem; font-weight: 700; margin-top: 2rem;">Modes</h2>

<h3 style="color: #5A3EED; font-weight: 700; margin-top: 1.5rem; margin-bottom: 0.6rem;">Analysis Mode</h3>

<p style="color: #555; line-height: 1.85; font-size: 1.02rem; margin-bottom: 0.6rem;">Given a circuit diagram, derive the Boolean expression:</p>

<ol style="padding-left: 1.2rem; margin: 0.5rem 0 1rem 0;">
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;">Identify each gate's function</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;">Write intermediate expressions</li>
<li style="margin-bottom: 0; line-height: 1.75; color: #333;">Combine to get final expression</li>
</ol>

<h3 style="color: #5A3EED; font-weight: 700; margin-top: 1.5rem; margin-bottom: 0.6rem;">Synthesis Mode</h3>

<p style="color: #555; line-height: 1.85; font-size: 1.02rem; margin-bottom: 0.6rem;">Given a Boolean expression, design the circuit:</p>

<ol style="padding-left: 1.2rem; margin: 0.5rem 0 1rem 0;">
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;">Identify required operations</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;">Determine needed gates</li>
<li style="margin-bottom: 0; line-height: 1.75; color: #333;">Connect gates appropriately</li>
</ol>
