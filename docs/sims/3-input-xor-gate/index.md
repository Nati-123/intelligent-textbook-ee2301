Interactive 3-input XOR gate demonstration with an 8-row truth table showing that the output is 1 when an odd number of inputs are 1.

## Learning Objective

Students will be able to extend the 2-input XOR gate concept to multiple inputs and understand the odd-parity behavior.

## Bloom Level

Understand (L2) - Explain, demonstrate

## Usage

Click the toggle boxes next to inputs A, B, and C. The truth table highlights the current row.
