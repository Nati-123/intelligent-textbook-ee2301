# Binary-Gray Code Converter

An interactive converter between Binary and Gray code representations.

## Overview

This MicroSim demonstrates the conversion between standard binary code and Gray code, where adjacent values differ by only one bit.

## Learning Objectives

- **Understand** Gray code properties (single-bit changes) (Bloom Level: L2)
- **Apply** XOR operations for code conversion (Bloom Level: L3)

## How to Use

1. **Enter** a binary number using the toggle buttons
2. **View** the equivalent Gray code
3. **Switch** between Binary→Gray and Gray→Binary conversion
4. **Observe** the XOR operations used in conversion
