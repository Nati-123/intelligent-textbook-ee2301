<h2 style="color: #5A3EED !important; border-bottom: 2px solid #5A3EED; padding-bottom: 0.3rem; font-weight: 700; margin-top: 2rem;">Description</h2>

<p style="color: #555; line-height: 1.85; font-size: 1.02rem; margin-bottom: 1.2rem;">

<div style="background: #FFF7DD; border: 2px solid #F0D87A; border-radius: 12px; padding: 20px 24px; margin: 1rem 0; box-shadow: 0 2px 8px rgba(184,134,11,0.08);">
<p style="color: #B8860B; font-weight: 700; margin-top: 0; margin-bottom: 8px;">F = x' &middot; F|<sub>x=0</sub> + x &middot; F|<sub>x=1</sub></p>
<ul style="list-style: none; padding-left: 0; margin: 0;">
<li style="margin-bottom: 0.3rem; line-height: 1.75; color: #333; padding-left: 1.4rem; text-indent: -1.4rem;"><span style="color: #B8860B; font-weight: 700; margin-right: 0.4rem;">&#9679;</span> <strong>F|<sub>x=0</sub></strong> is the cofactor when x=0 (substitute x=0 in F)</li>
<li style="margin-bottom: 0; line-height: 1.75; color: #333; padding-left: 1.4rem; text-indent: -1.4rem;"><span style="color: #B8860B; font-weight: 700; margin-right: 0.4rem;">&#9679;</span> <strong>F|<sub>x=1</sub></strong> is the cofactor when x=1 (substitute x=1 in F)</li>
</ul>
