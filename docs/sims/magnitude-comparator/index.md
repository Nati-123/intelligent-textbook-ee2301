# Magnitude Comparator

An interactive 4-bit magnitude comparator that determines the relationship between two binary numbers.

## Overview

This MicroSim demonstrates how a magnitude comparator compares two binary numbers and outputs which is greater, equal, or less.

## Learning Objectives

- **Understand** magnitude comparison of binary numbers (Bloom Level: L2)
- **Apply** comparator outputs for conditional logic (Bloom Level: L3)

## How to Use

1. **Set** the value of number A using the toggle buttons
2. **Set** the value of number B
3. **Observe** the three outputs: A>B, A=B, A<B
