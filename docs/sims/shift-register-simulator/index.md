# Shift Register Simulator

An interactive 4-bit shift register demonstrating serial and parallel data operations.

## Overview

This MicroSim demonstrates shift register operation including serial-in/serial-out and parallel load modes.

## Learning Objectives

- **Understand** shift register data movement (Bloom Level: L2)
- **Apply** shift registers for serial-to-parallel conversion (Bloom Level: L3)

## How to Use

1. **Enter** serial data bit by bit
2. **Click "Shift"** to move data through the register
3. **Use "Parallel Load"** to load all bits at once
4. **Observe** data movement through the flip-flops
