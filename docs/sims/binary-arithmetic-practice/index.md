## Description

Practice binary arithmetic operations with randomly generated problems. Learn the rules for binary addition and subtraction with carries and borrows.

## Learning Objectives

**Bloom Level**: Apply (L3)

- Perform binary addition with carry propagation
- Perform binary subtraction with borrowing
- Apply binary arithmetic rules correctly

## How to Use

1. Select addition or subtraction
2. Try to solve the problem mentally
3. Click "Show Answer" to check your work
4. Click "New Problem" for another challenge
