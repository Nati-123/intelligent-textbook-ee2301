# D Flip-Flop Simulator

An interactive edge-triggered D flip-flop demonstrating sampling at clock edges.

## Overview

This MicroSim shows how a D flip-flop captures the D input value only at the rising clock edge and holds it until the next edge.

## Learning Objectives

- **Understand** edge-triggered operation vs level-sensitive latches (Bloom Level: L2)
- **Apply** D flip-flop behavior to predict output sequences (Bloom Level: L3)

## How to Use

1. **Set** the D input value
2. **Click "Clock Pulse"** to generate a rising edge
3. **Observe** Q capturing D's value at the edge
4. **Note** that D changes between edges don't affect Q
