# SR Latch Simulator

An interactive SR latch demonstrating set, reset, hold, and invalid states.

## Overview

This MicroSim shows the fundamental memory element - the SR (Set-Reset) latch built from cross-coupled NOR gates.

## Learning Objectives

- **Understand** SR latch operation and feedback (Bloom Level: L2)
- **Analyze** the invalid state condition when S=R=1 (Bloom Level: L4)

## How to Use

1. **Toggle** the S (Set) and R (Reset) inputs
2. **Observe** the Q and Q' outputs
3. **Notice** the invalid state when both S and R are high
