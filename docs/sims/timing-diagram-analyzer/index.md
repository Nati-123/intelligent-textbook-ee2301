# Timing Diagram Analyzer

An interactive tool for analyzing flip-flop timing diagrams and predicting outputs.

## Overview

This MicroSim helps visualize and analyze timing diagrams showing clock, input, and output relationships for flip-flops.

## Learning Objectives

- **Apply** timing diagram analysis to predict flip-flop outputs (Bloom Level: L3)
- **Analyze** setup/hold time relationships (Bloom Level: L4)

## How to Use

1. **View** the clock and D input waveforms
2. **Identify** rising clock edges
3. **Trace** the D value at each edge to predict Q
