# FSM Designer

An interactive finite state machine simulator showing Moore machine behavior.

## Overview

This MicroSim demonstrates a simple Moore state machine with visual state diagram and transition animation.

## Learning Objectives

- **Understand** FSM state diagrams and transitions (Bloom Level: L2)
- **Analyze** Moore machine output based on current state (Bloom Level: L4)

## How to Use

1. **View** the state diagram
2. **Click "Input 0"** or **"Input 1"** to apply inputs
3. **Watch** the state transitions
4. **Observe** how outputs depend only on current state (Moore)
