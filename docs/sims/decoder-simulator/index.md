# Decoder Simulator

An interactive 2-to-4 decoder demonstrating binary to one-hot conversion.

## Overview

This MicroSim shows how a decoder converts a binary input code into a one-hot output where exactly one output is active based on the input combination.

## Learning Objectives

- **Understand** decoder operation and one-hot encoding (Bloom Level: L2)
- **Apply** decoder outputs for function implementation (Bloom Level: L3)

## How to Use

1. **Toggle** the binary inputs A1, A0
2. **Observe** which output (Y0-Y3) becomes active
3. **View** the truth table showing all input/output combinations
