## Description

This interactive force-directed graph shows how all 13 units in the EE2301 course connect and build upon each other. Drag nodes to explore relationships. Click on a unit to see its prerequisite and dependent connections highlighted.

## Learning Objectives

**Bloom Level**: Evaluate (L5)

- Evaluate how course topics build upon each other
- Identify prerequisite chains and dependencies
- Understand the progression from fundamentals to integration
- Synthesize a holistic view of digital system design

## How to Use

1. **Drag** unit nodes to rearrange the graph layout
2. **Click** a unit to highlight its connections
3. Observe **arrow directions** showing prerequisite relationships
4. **Hover** over connections to see the relationship description
5. The graph self-organizes using force-directed layout
