<h2 style="color: #5A3EED !important; border-bottom: 2px solid #5A3EED; padding-bottom: 0.3rem; font-weight: 700; margin-top: 2rem;">Description</h2>

<p style="color: #555; line-height: 1.85; font-size: 1.02rem; margin-bottom: 1.2rem;">
This MicroSim provides an interactive demonstration of a 3-input AND gate. The simulation displays the standard AND gate symbol with three inputs (A, B, C), clickable toggle buttons, and a complete eight-row truth table.
</p>

<p style="color: #555; line-height: 1.85; font-size: 1.02rem; margin-bottom: 1.2rem;">
The 3-input AND gate outputs 1 only when all three inputs are 1. For every other combination, the output is 0. As you toggle inputs, the truth table row corresponding to the current input combination is highlighted in real time.
</p>

<h2 style="color: #5A3EED !important; border-bottom: 2px solid #5A3EED; padding-bottom: 0.3rem; font-weight: 700; margin-top: 2rem;">Lesson Plan</h2>

<h3 style="color: #5A3EED; font-weight: 700; margin-top: 1.5rem; margin-bottom: 0.6rem;">Before the Simulation (5 minutes)</h3>

<ul style="list-style: none; padding-left: 0.8rem; margin: 0.5rem 0 1rem 0;">
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><span style="color: #5A3EED; font-weight: 700; margin-right: 0.5rem;">&#9679;</span> Review the 2-input AND gate and its truth table</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><span style="color: #5A3EED; font-weight: 700; margin-right: 0.5rem;">&#9679;</span> Ask students how many rows the truth table would have for 3 inputs (2^3 = 8)</li>
<li style="margin-bottom: 0; line-height: 1.75; color: #333;"><span style="color: #5A3EED; font-weight: 700; margin-right: 0.5rem;">&#9679;</span> Introduce the idea that AND generalizes to any number of inputs</li>
</ul>

<h3 style="color: #5A3EED; font-weight: 700; margin-top: 1.5rem; margin-bottom: 0.6rem;">During the Simulation (15 minutes)</h3>

<ol style="padding-left: 1.2rem; margin: 0.5rem 0 1rem 0;">
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;">Start with all inputs at 0 and verify the output is 0</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;">Toggle inputs one at a time to explore different combinations</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;">Find the single combination that produces output 1 (A=1, B=1, C=1)</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;">Count how many of the 8 rows produce output 0 versus output 1</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;">Discuss why AND is sometimes called the "all-or-nothing" gate</li>
<li style="margin-bottom: 0; line-height: 1.75; color: #333;">Compare mentally with a 2-input AND to see the pattern</li>
</ol>

<h3 style="color: #5A3EED; font-weight: 700; margin-top: 1.5rem; margin-bottom: 0.6rem;">After the Simulation (5 minutes)</h3>

<ul style="list-style: none; padding-left: 0.8rem; margin: 0.5rem 0 1rem 0;">
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><span style="color: #5A3EED; font-weight: 700; margin-right: 0.5rem;">&#9679;</span> Ask students to predict the truth table for a 4-input AND gate</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><span style="color: #5A3EED; font-weight: 700; margin-right: 0.5rem;">&#9679;</span> Discuss how multi-input AND gates are used in address decoders</li>
<li style="margin-bottom: 0; line-height: 1.75; color: #333;"><span style="color: #5A3EED; font-weight: 700; margin-right: 0.5rem;">&#9679;</span> Connect to the NAND gate as the complement of AND</li>
</ul>
