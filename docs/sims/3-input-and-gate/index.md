# 3-Input AND Gate

Interactive 3-input AND gate demonstration with an 8-row truth table showing that the output is 1 only when all three inputs are 1.

## Learning Objective

Students will be able to extend the 2-input AND gate concept to multiple inputs and predict outputs for any 3-input combination.

## Bloom Level

Understand (L2) - Explain, demonstrate

## Usage

Click the toggle boxes next to inputs A, B, and C. The truth table highlights the current row.
