<h2 style="color: #5A3EED !important; border-bottom: 2px solid #5A3EED; padding-bottom: 0.3rem; font-weight: 700; margin-top: 2rem;">Laws Covered</h2>

<ol style="padding-left: 1.2rem; margin: 0.8rem 0 1.2rem 0;">
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><strong>Identity</strong>: A·1 = A, A+0 = A</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><strong>Null</strong>: A·0 = 0, A+1 = 1</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><strong>Idempotent</strong>: A·A = A, A+A = A</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><strong>Complement</strong>: A·A' = 0, A+A' = 1</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><strong>Double Negation</strong>: (A')' = A</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><strong>Commutative</strong>: A·B = B·A, A+B = B+A</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><strong>Associative</strong>: (A·B)·C = A·(B·C)</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><strong>Distributive</strong>: A·(B+C) = A·B + A·C</li>
<li style="margin-bottom: 0.5rem; line-height: 1.75; color: #333;"><strong>Absorption</strong>: A + A·B = A</li>
<li style="margin-bottom: 0; line-height: 1.75; color: #333;"><strong>De Morgan's</strong>: (A·B)' = A' + B'</li>
</ol>
