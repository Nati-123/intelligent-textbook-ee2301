## Description

Explore how different number systems relate to each other. Click on concepts to learn about binary, decimal, octal, hexadecimal, and signed number representations.

## Learning Objectives

**Bloom Level**: Understand (L2)

- Explain relationships between different number bases
- Describe how binary relates to octal and hexadecimal
- Understand the role of signed vs unsigned representations

## How to Use

1. Hover over any concept node to see its description
2. Click a node to keep its information displayed
3. Follow the arrows to understand relationships
4. Notice how binary is central to digital systems
