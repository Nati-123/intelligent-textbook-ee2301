## Description

This MicroSim breaks down how positional notation works by showing each digit's position, the corresponding power of the base, and how each digit contributes to the total value.

## Learning Objectives

**Bloom Level**: Understand (L2)

- Explain how positional notation represents numerical values
- Calculate the decimal equivalent of a number in any base
- Identify the place value of each digit position

## How to Use

1. Enter a number in the input field
2. Select the base (binary, octal, decimal, or hex)
3. Observe the breakdown of positions, powers, and contributions
