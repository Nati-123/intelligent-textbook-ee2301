# K-map Practice Challenge

<iframe src="main.html" height="550px" width="100%" scrolling="no"></iframe>


## Description

Test your K-map simplification skills with practice challenges. Each challenge presents a Boolean function as minterms - find the minimal SOP expression.

## Learning Objectives

**Bloom Level**: Apply (L3)

- Apply K-map grouping rules
- Derive minimal Boolean expressions
- Practice pattern recognition in K-maps

## Challenges

1. **Challenge 1**: 3-variable, column pattern
2. **Challenge 2**: 3-variable, complementary pattern
3. **Challenge 3**: 3-variable, majority function
4. **Challenge 4**: 4-variable, corner wrap-around
5. **Challenge 5**: 4-variable, half-map pattern

## Tips

- Look for the largest groups first
- Remember wrap-around groupings
- Use hints if stuck
- Check your answer before revealing

## How to Use

1. Examine the K-map and identify the minterms
2. Find groups (pairs, quads, octets)
3. Determine the simplified expression
4. Click "Show Answer" to check
5. Navigate through all challenges
