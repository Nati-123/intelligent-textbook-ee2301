# 3-Input OR Gate

Interactive 3-input OR gate demonstration with an 8-row truth table showing that the output is 1 when any input is 1.

## Learning Objective

Students will be able to extend the 2-input OR gate concept to multiple inputs and predict outputs for any 3-input combination.

## Bloom Level

Understand (L2) - Explain, demonstrate

## Usage

Click the toggle boxes next to inputs A, B, and C. The truth table highlights the current row.
