# Course Structure Tree

An interactive tree diagram visualizing the complete structure of the Digital System Design textbook.

## Overview

This MicroSim provides a hierarchical view of all course content, including:

- **10 Units** covering digital system design fundamentals
- **Unit Content** pages with detailed educational material
- **Quizzes** for each unit with multiple-choice questions
- **References** with annotated academic sources
- **37 MicroSims** for interactive learning
- **Resources** including FAQ, Glossary, and Learning Graph

## How to Use

1. **Click** on any node with children to expand or collapse it
2. **Scroll** with mouse wheel to navigate the tree
3. Use **Expand All** to see the complete structure
4. Use **Collapse All** to return to the overview

| Color | Type |
