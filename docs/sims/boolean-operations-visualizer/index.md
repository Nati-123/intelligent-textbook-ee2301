## Description

Explore Boolean operations using interactive Venn diagrams. Toggle input values and see how AND, OR, NOT, NAND, NOR, and XOR operations work visually.

## Learning Objectives

**Bloom Level**: Understand (L2)

- Explain how each Boolean operation works
- Interpret Venn diagram representations of logic operations
- Predict outputs for given Boolean inputs

## How to Use

1. Select a Boolean operation from the dropdown
2. Click the A and B toggles to change input values
3. Observe the highlighted regions in the Venn diagram
4. Check the mini truth table for all input combinations
