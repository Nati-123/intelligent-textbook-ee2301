# LinkedIn Announcement

## Full Version

---

Excited to share a project I've been building: an open-source, AI-assisted interactive textbook for EE 2301 — Introduction to Digital System Design!

This intelligent textbook is designed for sophomore and junior-level Electrical and Computer Engineering students at the University of Minnesota — Twin Cities. Built with MkDocs Material and Claude Code, it provides structured, scaffolded learning with concept dependency graphs, interactive simulations, and comprehensive assessments.

By the numbers:
- 13 units covering number systems through system integration
- 410 concepts organized in a learning dependency graph
- 110 interactive MicroSims (p5.js simulations) you can run in your browser
- 4,394 LaTeX/MathJax equations
- 353 practice and challenge problems with detailed solutions
- 130 quiz questions across Bloom's Taxonomy levels
- 398 glossary terms with ISO 11179-compliant definitions
- 76 FAQs for quick reference
- 204,000+ words (~877 equivalent printed pages)

This project demonstrates how AI can augment educational content creation while maintaining pedagogical quality and rigor. All content is generated using Claude Code with human oversight, following Bloom's Taxonomy for learning outcomes.

Explore the full textbook here: https://Nati-123.github.io/intelligent-textbook-ee2301/
