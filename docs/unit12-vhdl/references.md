<li style="margin-bottom: 1rem;"><strong style="color: #333;">VHDL for Engineers</strong> — Kenneth L. <li style="margin-bottom: 1rem;"><a href="https://ghdl.github.io/ghdl/" style="color: #5A3EED; font-weight: 600;">GHDL Open-Source VHDL Simulator</a> — GHDL Project — Open-source VHDL analyzer, compiler, and simulator supporting IEEE 1076 standards, useful for learning and verifying VHDL designs without commercial tools.</li>
