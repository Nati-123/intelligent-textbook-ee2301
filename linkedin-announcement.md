# LinkedIn Announcement

## Full Version

Excited to share my senior design project: an AI-assisted interactive textbook for EE 2301 — Introduction to Digital System Design at the University of Minnesota!

This intelligent textbook covers the full digital logic curriculum from number systems and Boolean algebra through sequential circuits, programmable logic devices, VHDL, and system integration. Built using MkDocs Material with Claude Code for AI-assisted content generation, it features interactive simulations, concept dependency graphs, and comprehensive assessment tools.

By the numbers:
- 13 units covering the complete EE 2301 curriculum
- 410 concepts organized in a learning dependency graph
- 110 interactive MicroSims (p5.js simulations you can run in your browser)
- 269 practice problems + 84 challenge problems with detailed solutions
- 130 multiple-choice quiz questions aligned to Bloom's Taxonomy
- 398 glossary terms with ISO 11179-compliant definitions
- 4,300+ LaTeX equations
- 203,000+ words (~877 equivalent printed pages)

Every unit includes content, quizzes, practice problems, challenge problems, a glossary, and curated references. The interactive MicroSims let students explore concepts hands-on — from logic gate simulators and K-map solvers to FSM designers and VHDL synthesis tools.

Generated using Claude AI and the intelligent textbook framework, this open-source project demonstrates how AI can augment educational content creation while maintaining pedagogical quality and technical accuracy.

Explore the full textbook: https://Nati-123.github.io/intelligent-textbook-ee2301/

Source code: https://github.com/Nati-123/intelligent-textbook-ee2301
